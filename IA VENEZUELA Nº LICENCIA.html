ℹ️ Guía de Análisis Forense

Esta herramienta realiza un análisis integral de las Licencias de Conducir de Venezuela, combinando Inteligencia Artificial, estimación matemática y análisis forense digital.

🤖 1. Extracción con IA (Gemini)

Al arrastrar una imagen o PDF (o usar Ctrl+V), la IA localiza y extrae automáticamente el Nro De Verificación y la F. Expedición del documento.

📈 2. Lógica de Progresión Aritmética

El sistema compara el número introducido con nuestra Base de Datos local. Analiza el registro inmediatamente anterior y posterior para calcular una Fecha Estimada matemática. Si la fecha real del documento no cuadra con esta estimación cronológica, el sistema lo marcará con una Alerta de Incongruencia / Posible Falso.

🔍 3. Análisis de Metadatos (Trazabilidad PDF)

  • Editor Gráfico: Detecta si fue guardado con herramientas de manipulación (Word, Photoshop, Canva, Corel, Illustrator...). Si es así, lanza una Alerta de Documento Falso.
  • Fechas Internas: Compara la fecha de creación del archivo con su fecha de modificación. Si difieren, avisa de una posible alteración.
  • Veredicto: Si no hay software comercial ni anomalías en las fechas, lo califica como Documento Indubitado.

📸 4. Extracción Profunda (Render)

Nuestro servidor lee el código crudo del PDF y extrae las capas y objetos ocultos (XObjects). Esto nos permite sacar la fotografía del titular, el QR y los fondos sin filtros, indicando sus dimensiones exactas (px) y su peso (KB) para detectar manipulaciones gráficas.

Manual de Usuario y Ayuda

⚙️ Configuración API Gemini

Para el correcto funcionamiento del análisis de imagen:

  • Modelo obligatorio: Debes usar siempre gemini-2.5-flash. Es el único probado y funcional para esta tarea.
  • Si falla la API: Si cambias de navegador o la IA deja de responder (error de cuota o clave inválida), pulsa el botón OBTENER CLAVE API en los ajustes para ir a tu listado privado, copia una nueva y pégala en el campo correspondiente.

🔢 Introducción Manual

Puedes escribir el número de serie en el buscador principal:

  • El sistema te dirá si el número es coherente basándose en registros anteriores y posteriores.
  • Si el número ya existe, te permitirá editarlo si detectas algún error.
  • Si no existe, podrás añadirlo manualmente.

📂 Organización Automática

Los registros se guardan automáticamente en su listado correspondiente:

  • Serie A: Documentos estándar.
  • Serie L: Licencias que empiezan por la letra L.

🤖 Arrastrar y Soltar (IA)

Puedes arrastrar una imagen del permiso directamente al recuadro:

  • El sistema extraerá automáticamente: Número de Serie, Vilaya (traducida) y Fecha de Expedición.
  • Advertencia: La API es gratuita. No arrastres muchos documentos de golpe (hazlo uno a uno) o el sistema podría saturarse y dejar de funcionar temporalmente.

Buscador Inteligente

Escribe cualquier término (País, Tipo, Año) para filtrar al instante. El sistema busca coincidencias en todas las etiquetas del documento.

Filtros Rápidos

Visor Premium

Haz clic en el icono del ojo para abrir el documento. Puedes usar las flechas laterales para navegar entre los archivos de ese mismo país sin salir del visor.

Lógica de la Auditoría

⚠️ REGLA DE ORO: El nombre del archivo debe EMPEZAR exactamente por el tipo (Ej: "TIE 2024..."). Si empieza por "Solicitud TIE", el sistema lo ignorará para evitar errores.

El Sistema Semáforo

  • ● VERDE: Indica que el mes/semestre está cubierto. El sistema detecta el patrón AAAA-MM en el nombre.
  • ● ROJO: Indica que el documento falta en la base de datos para ese periodo.

¿Qué muestra el PDF?

El reporte PDF genera un historial completo desde el primer año registrado de cada país. Mostrará los años con faltas detallando cada mes en color rojo/verde.

Gestionar Reglas

Para añadir una regla: Escribe el tipo (ej: PNC), elige la frecuencia y dale a Guardar. El sistema empezará a auditar ese tipo de inmediato en todos los países.
